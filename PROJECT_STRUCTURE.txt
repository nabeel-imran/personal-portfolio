Getting Started
===============

1. Install dependencies:
   npm install

2. Configure .env.local:
   cp .env.local.example .env.local
   (edit and add your keys)

3. Run development server:
   npm run dev

4. Visit:
   http://localhost:3000

For detailed instructions, see: GET_STARTED.md

Quick Commands
==============
