# Project Images

Place your project screenshots and images in this directory.

## Recommended Specifications

- **Format**: JPG or PNG
- **Dimensions**: 1200x800px (3:2 ratio)
- **File Size**: < 500KB (optimize for web)
- **Naming**: Use kebab-case (e.g., `rankguide.jpg`, `ai-image-gen.png`)

## Required Images

1. `rankguide.jpg` - RankGuide SEO Analytics Tool
2. `background-check.jpg` - Global Background Check Automation
3. `ai-image-gen.jpg` - AI Image Generation Platform
4. `ecommerce.jpg` - E-Commerce Automation Suite
5. `web-scraper.jpg` - Smart Web Scraping Framework
6. `chatbot.jpg` - AI Chatbot Platform
